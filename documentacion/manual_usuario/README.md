# Manual de usuario

Guía para usuarios con capturas de pantalla y pasos para usar el sistema.

## Reglas de uso por perfil
- Si tu perfil es Jugador, podrás ver torneos y unirte a actividades, pero no podrás crear torneos.
- Si tu perfil es Organizador o Administrador, podrás crear y administrar torneos desde la pantalla de torneos.
