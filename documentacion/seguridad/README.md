# Documentación de seguridad

Análisis OWASP, políticas, gestión de contraseñas y riesgos.

## Control de acceso por roles
- El acceso a la creación de torneos está restringido a usuarios con rol de Organizador o Administrador.
- Los usuarios con rol de Jugador no deben poder organizar torneos, incluso si intentan acceder a la ruta de creación manualmente.
