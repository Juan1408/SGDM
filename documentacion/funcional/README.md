# Documentación funcional

Contiene casos de uso, historias de usuario y especificaciones funcionales.

## Reglas de acceso por rol
- Jugador/Participante: puede consultar torneos, inscribirse y participar en equipos.
- Organizador: puede crear y administrar torneos.
- Administrador: puede supervisar la plataforma y también gestionar torneos.
- La creación de torneos no está habilitada para usuarios con rol de jugador.
